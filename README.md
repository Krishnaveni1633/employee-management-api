
# Employee Management System API

Full stack Employee Management System with AI Document Summarization, JWT authentication, and React frontend

## 🌍 Live API
**Base URL:** https://employee-management-api-59jd.onrender.com

**API Documentation:** https://employee-management-api-59jd.onrender.com/docs

## 🚀 Tech Stack
- Python FastAPI
- PostgreSQL
- SQLAlchemy
- JWT Authentication
- bcrypt
- Docker
- Render (Cloud Deployment)

- [x] Search & Pagination
- [x] AI Document Summarizer
- [x] Email Notifications
- [x] Docker
- [x] Cloud Deployment
